restaurent Click css HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It defines the layout and meaning of content—such as text, images, and links—using a system of tags and elements. It is the foundational, static structure of web development, often enhanced with CSS for styling and JavaScript for interactivity.

CSS (Cascading Style Sheets) is a stylesheet language used to define the visual presentation and layout of a document written in a markup language like HTML or XML. It is a cornerstone technology of the World Wide Web, working alongside HTML (which provides structure) and JavaScript (which provides interactivity).
